# Что выведет программа? Объяснить вывод программы.

````
package main

func main() { 
    ch := make(chan int) 
    go func() { 
        for i := 0; i < 10; i++ { 
            ch <- i 
            } 
    }()

    for n := range ch {
    println(n)
    }
} 
````

````
Ответ:

Программа выведет цифры от 0 до 9 и дальше впадет в deadlock. 
Это вызвано тем, что в канал ch больше не пишутся значения, однако мы продолжаем ждать их, чтобы считать. 
Т.о. происходит блокировка горутины, т.к. канал не закрыт и значения больше не будут туда записываться.
````
