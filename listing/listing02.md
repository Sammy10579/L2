# Что выведет программа? Объяснить вывод программы.

````
package main

import ( "fmt" )

func test() (x int) { 
    defer func() { 
        x++ }()
        x = 1 return 
}

func anotherTest() int { 
    var x int defer func() { 
        x++ 
        }() 
        x = 1 
        return x 
}

func main() { 
    fmt.Println(test()) fmt.Println(anotherTest()) 
} 
````

````
Ответ:

Будет выведено 2 и 1.

В первом случае выведется 2, потому что в функции явно указано, что должен вернуться x. 
Т.к. defer выполняется перед выходом из функции, то после того, как функция вернет х, defer увеличит этот х на 1

Во втором случае выведется 1, потому что в функции указано, что она должна вернуть инт, 
но без явного указания переменной, которая возвращается. Таким образом, вернется конкретное значение, 
а уже после defer увеличит х
````
